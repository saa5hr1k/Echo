# Echo

**Echo** is an AI-powered learning platform that transforms any topic into a complete, personalized learning experience. Instead of relying on pre-built courses, Echo generates an entire curriculum on demand, including AI-written textbooks, lessons, quizzes, flashcards, practice exercises, and adaptive learning paths.

Whether you're learning Mathematics, Programming, Astrophysics, Business, or any custom subject, Echo acts as your personal tutor, textbook publisher, and study coach.

✨ Features

* 🤖 AI-generated courses for any subject
* 📚 Professionally structured AI textbooks
* 🧠 Adaptive AI tutor with personalized guidance
* 📝 AI-generated quizzes, flashcards, and revision notes
* 🎯 Personalized learning paths based on your progress
* ⭐ Gamified learning with XP, levels, achievements, and streaks
* 🌳 Visual skill trees and mastery tracking
* 📖 Digital library to revisit every generated course
* 📱 Modern, responsive, and intuitive interface

🚀 Vision

Echo's mission is to make high-quality education accessible to everyone by creating a personalized AI-powered university that adapts to every learner. Our goal is to make mastering any subject engaging, effective, and enjoyable.

🛠️ Tech Stack

* **Frontend:** Next.js, TypeScript, Tailwind CSS, Framer Motion
* **Backend:** Next.js API Routes, Prisma, PostgreSQL
* **Authentication:** Clerk
* **AI:** OpenAI & Anthropic APIs
* **Deployment:** Vercel
